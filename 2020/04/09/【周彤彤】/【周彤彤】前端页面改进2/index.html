    //将总数据中sumArrays每一列数组分组出来形成sumArray=[oneArray[oneColumn,...],twoArray[twoColumn,...],...,sevenArray[sevenColumn,...]]

                 var arrays2=sumArray[t];//总共有7列，当前第t，单个数组

                     var element = arrays2[i]; //当前七列中第t列数组当中具体的元素
                     if (number == element) {
                         if(map.get(element)==null || map.get(element)==undefined){

                        }
                     }

                     for(var n=0;n<arrays2.length;n++){
                         var key=arrays2[n]; //当前七列中第t列数组当中具体的元素
                         if(key!=null && key==element && number == key){ //如果存在有当前循环的数组当中的取值与map当中的键相同的则重新赋值，重新赋值后进行break;//var y=i-n;
                             sumList.push(n);

                             //存储每一个元素他的前一位相同元素的下标；y为当前元素的下标
                             if((map.get(element)!=null && map.get(element)!=undefined)
                                     && map.get(element).prior==null && map.get(element).curr==null && map.get(element).size==null){
                                 map.put(element,{prior: n, curr: n, size: 0}); //第一次遇到元素相同的时候
                             }
                             if((map.get(element)!=null && map.get(element)!=undefined)
                                     && (map.get(element).prior!=null && map.get(element).curr!=null && map.get(element).size!=null)
                                      && map.get(element).prior==map.get(element).curr){
                                 map.put(element,{prior: map.get(element).prior, curr: n, size: n-map.get(element).prior}); //第二次遇到元素相同的时候
                             }
                             if((map.get(element)!=null && map.get(element)!=undefined)
                                     && (map.get(element).prior!=null && map.get(element).curr!=null  && map.get(element).size!=null)
                                     && map.get(element).prior!=map.get(element).curr) {
                                  // 此时 map 中的 curr 即当前 key 的相同元素的上一位置 prior 的 index ;取出 prior;
                                 // 得到当前 key index 与当前 key index 的 prior 的 size；
                                 // 如果当前 size较之此时map当中element元素的size要大，则进行替换
                                 var currPrior=map.get(element).curr; //当前的上一位
                                 var currSize=n-currPrior; //当前的大小
                                 var priorSize=map.get(element).size; //上一位的大小
                                 if(currSize>priorSize){
                                     map.put(element,{prior:currPrior, curr:n, size:currSize});
                                 }
                             }
                         }else{
                             continue;//跳出对于第i列数组中element元素对于当前n数组的当次循环，进入当前n数组迭代下一元素key的循环
                         }
                     }
                 }
                 console.log("第"+t+"列当中，每一个元素的单排最大遗漏数："+JSON.stringify(map));
                 if(map.get(number)!=null || map.get(number)!=undefined){
                     //取值则为当前该map.get(number)即当前列当中的单排最大遗漏数
                     singleMap.put(t,map.get(number).size-1);//singleMap.put(t,map.get(number)-1);//key t：哪一列， value 当前该数值在当前排t的最大遗漏数  map.get(number)需要减去1
                 }else{
                     //当前列取出number如果为空则说明当前列当中不存在有该数值，则替换为0
                     singleMap.put(t,'当前列不存在'+number);
                 }
             }
             //遍历完7个数组后得到当前查询的该数值的所有index
             //得到当前查询的该数值的在7列中的所有index，并进行index排序sort,每一个index都与index.next进行比较得到size

                     }else{
                         var nextSize=sumList[g+1]-sumList[g];
                         if(nextSize>sumMaxNumber){
                            sumMaxNumber=nextSize;
                        }
                     }
                 }
             }
             var mapArrays=singleMap.entrys();//循环之后获取得到singleMap,即单排数
             if (h == 0) { //即第一个数值的时候填入行tr:arrays.length+2
                 td.innerHTML="数字: "+number+" | 单排最大遗漏数 | 总排最大遗漏数: "+sumMaxNumber;
                 for(var r=0;r<mapArrays.length;r++) {
                     var td22 = tr.insertCell(r+1);//当前行处于第一行
                     td22.innerHTML = singleMap.get(r);
                 }
                 tr.insertCell(mapArrays.length+1);
             } else {
                 //新增行并插入
                 var newTr=detailData.insertRow(arrays.length+2+h);
                 var newTd=newTr.insertCell(0);
                 newTd.innerHTML="数字: "+number+" | 单排最大遗漏数 | 总排最大遗漏数: "+sumMaxNumber;
                 newTd.style.fontWeight="bolder";
                 newTd.colSpan=2;
                 for(var r=0;r<mapArrays.length;r++) {
                     var td22 = newTr.insertCell(r+1);//从第二行开始进行新增列
                     td22.innerHTML = singleMap.get(r);
                 }
                 newTr.insertCell(mapArrays.length+1);
             }
             console.log("================== 查询数值："+number+" END ==================");
         }
     }
