  1. 单机MySQL的美好年代

在90年代，一个网站的访问量一般都不大，用单个数据库完全可以轻松应付。
在那个时候，更多的都是静态网页，动态交互类型的网站不多。

上述架构下，来看看数据存储的瓶颈是什么？

  1. 数据量的总大小 —-》一个机器放不下时
  2. 数据的索引（B+Tree）—-》一个机器的内存放不下时
  3. 访问量（读写混合） —-》一个实例不能承受时
